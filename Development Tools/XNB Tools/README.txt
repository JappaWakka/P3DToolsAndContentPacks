XNB Image Converter & XNB Sound Converter
--------
Convert XNB Image and Sound files to their original file format.

Usage:
* Put the XNB files in the same folder as the converter (or the other way around)
* Open the executable and type in the name of the XNB file without the extension (.xnb)
* Press the button and you're done!
========

XNB Builder
--------
Converts content files to XNB format. This is pretty much only necessary for getting 3D Models (in FBX file format) into Pok√©mon 3D.

Usage:
* Open XNB Builder Download Link
* Download both XNBBuilder.zip & XnaForVS2022.zip on the linked page
* Read the instructions on what to install before you can use the software
* After installing everything, open xnbbuilder.exe
* Browse to the source files and select the ones you want to convert (you can select multiple files at the same time)
* Change the output directory if you want
* Press Convert and after a moment a pop-up will show up and you'll have your XNB files.
